Working with RSS?

Me too. JaxbRss offers a simple data model layer that encompasses the most commonly used RSS fields. What's more, it can be serialized to a number of different formats.

Podcasting Data Model = Easy

Podcasting can be confusing. There are several different specificaitons that authors often want to integrate into a single feed for maximum interoperability. No worries - this data model covers base rss, atom, and itunes.
